# Simple clicker game with the integration of Yandex SDK


Stack: C#, Unity2D, Git, Visual Studio, Figma, Yandex Games

* Developed the whole game as an independent single developer
* Conducted market research on clickers on the platform, similar games on other platforms, and the mathematics of IDLE games
* Compiled a concept document and decomposed it into tasks in the task tracker
* Created and maintained a flowchart and a UML class diagram
* Created UI Wireframe and later implemented in Unity
* Drawn the art by myself, using both 2D art software and Stable Diffusion

The game was available in the app store: https://yandex.ru/games/app/253227 (was removed due to update inactivity)
